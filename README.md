# FieldCraft

FieldCraft is a structured, skills-first outdoor survival manual focused on practical fieldcraft.

This is not a prepper guide. It emphasizes:
- decision-making under pressure
- efficient use of time and energy
- survival with minimal equipment
- repeatable skill development

## Doctrine

- Primary: Scouting / fieldcraft
- Overlay: Military survival priorities (shelter, water, fire, signaling, movement)
- Anchor: Search and Rescue (stay alive, be found)

## Structure

- Core survival framework
- Core field skills
- Movement and environment
- Sustainment skills
- Medical and human factors
- Training and execution
- Reference material
