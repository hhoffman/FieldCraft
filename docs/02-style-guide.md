# Style Guide

## Tone
- Direct and instructional
- No filler or motivational language
- Assume reader competence

## Writing Rules
- Prefer clarity over completeness
- Use short, precise sentences
- Avoid storytelling unless illustrating failure

## Content Bias
- Skills over gear
- Decision-making over description
- Efficiency over completeness

## Formatting
- Use headings consistently
- Use bullet points for procedures
- Avoid long paragraphs where structure improves clarity
